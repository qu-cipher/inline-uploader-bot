# quantum's media uploader bot

### How to use
1. install packages
```bash
yarn
# or
npm install
```

2. Grab your bot token from [@BotFather](https://t.me/BotFather) and set it at `src/bot.ts:29`

3. Build project
```bash
yarn build
# or
npm run build
```

4. Start project (Personal Computer / Server / Anywhere else)
```bash
yarn start
# or
npm start
```
